# 3D Treasure Hunt

![](images/forest_1.jpg)
A 3D adventure treasure hunt prototype developed in Unity.
The player explores a stylized environment, interacts with objects, and fights using combo attacks while being guided by a magical fairy companion.

The project focuses on exploration mechanics and visual atmosphere through shaders, particle systems, and environmental effects.

## Features

- 3D exploration gameplay
- Combat system with attack combos
- Destructible objects and environmental interactions
- Fairy companion follower system
- Stylized environment with atmospheric effects

## Technical Highlights

- Custom Shader Graph used to create the fairy’s liquid-like visual appearance
- Scripted follower movement system controlling the fairy companion behavior
- Particle systems for environmental effects such as tree leaves and butterflies
- Use of skyboxes and volumetric clouds to create immersive atmospheric lighting
- Implementation of character animations sequences and gameplay interactions
